1)Problem
Return the total number of digits in a given integer n.
Approach
If n == 0, return 1.
Otherwise, repeatedly divide the number by 10 and count how many times it can be divided until it becomes 0.
Time Complexity
O(log₁₀ n)
Space Complexity
O(1)
